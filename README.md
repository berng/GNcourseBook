	Книга представляет собой конспект лекций по графовым нейронным сетям и рекомендательным системам, 
	читаемым автором в магистратуре и бакалавриате на факультете бизнес-коммуникаций и информатики 
	Иркутского государственного университета. Материалы предназначены для студентов и специалистов, 
	уже знакомых с базовыми методами машинного обучения с учителем и без учителя.
	Работа выполнена при поддержке Фонда Потанина.
